# Automatic Emergency Light

## 🔋 Project Overview
This project is an Automatic Emergency Light system designed using KiCad. It automatically turns ON during power failure and uses battery backup.

## ⚙️ Features
- Automatic ON during power failure
- Battery backup system
- Efficient LED lighting
- Simple and cost-effective design

## 🛠️ Tools Used
- KiCad (Schematic + PCB Design)

## 📁 Files Included
- Schematic (.kicad_sch)
- PCB Layout (.kicad_pcb)
- Project files

## 🚀 Applications
- Homes
- Emergency systems
- Power outage backup lighting
